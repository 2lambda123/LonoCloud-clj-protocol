client-message-handler

(client-message-handler cfg buf)

Respond to DHCP messages. Read the DHCP message from buf, and depending on the message and configuration in cfg send an appropriate response. If buf is nil then this means we are responding to a trigger (i.e. send DISCOVER) rather than to an actual received message.
